# Restock!
Laravelアプリの学習用に作成した、日用品管理Webアプリです。
## 機能
- 新規利用登録及びログイン
- ストック数管理
- 消費ペース自動計算・表示
- 次回ストック開封予想
- ワンボタン開封・補充機能
- 検索機能
## 使い方
「ログイン」もしくは「新規登録」から新規利用登録
### ボタン説明
- 「新規登録」：管理するアイテムを追加する。
- 「開封」：ストック数を-1にして、開封日に現在の年月日を登録する。
- 「補充」：ストック数を+1する。
- 「削除」：アイテムを削除する。
- 「検索」：アイテムのカテゴリおよび品名を検索して絞り込み表示を行う。
- 「編集」：アイテムの行をクリックすると、アイテムの情報を編集できる。
